# Auralis - Music Streaming Web Application

## Project Overview

Auralis is a frontend-only music streaming web application inspired by Spotify. The application provides users with an intuitive interface to browse songs, discover artists, manage playlists, and enjoy a realistic music streaming experience.

## Technologies Used

* ReactJS
* TypeScript
* Vite
* React Router DOM
* Tailwind CSS
* Axios
* JSON Server (db.json)

## Prerequisites

- Node.js 18+

## Setup & Run Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the mock backend server:
   ```bash
   npx json-server --watch db.json --port 3001
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.
